Why/how did we choose this data?

In the Fall of 2015 CTWAG group members decided to use one common data set through which to explore computational text analysis methods. Members chose to focus our collective efforts on the Congressional Record. We chose the Congressional Record for the following reasons:

  1. "Big" data set: There is a large amount of data.
  2. Publicly available data: The data is public information, so there are no copyright issues to our work.
  3. Multiple forms of communication: The record includes both spoken and written text communication
  4. External linkages: Potential to link to voter records, meta data about congress people, committees served on, etc.
